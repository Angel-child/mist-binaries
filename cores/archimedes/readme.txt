Archimedes
==========

This is early beta version of a Acorn Archimedes core.

Needs at least firmware version 20190303!

Cores in the old directory are only usable with firmware before 20190303.

To use the IDE interface, at least firmware 20191102 is required.
 
For more details see 
https://github.com/mist-devel/mist-board/wiki/CoreDocArchimedes
